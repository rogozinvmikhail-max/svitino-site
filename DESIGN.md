# SVITINO.TV — DESIGN.md

Визуальная дизайн-система сайта. Выведена из фотографий самого Свитино (палитра — из реальных пикселей) и из истории места. Любые правки сайта должны оставаться в этой системе.

## 1. Суть и нерв бренда

> **С 1945 года между двух дубов в Свитино светил экран. Сегодня это SVITINO.TV.**

После Великой Отечественной войны, с 1945 до начала 1950-х, по выходным в деревню приезжал киномеханик, натягивал экран между двух дубов и показывал жителям кино («Броненосец Потёмкин», «Путёвка в жизнь», «Чапаев»). Это было первое знакомство деревни с кинематографом. SVITINO.TV — прямое продолжение того первого экрана, спустя 80 лет.

**Характер:** живое, тёплое, человеческое — и при этом современное, технологичное. Природа + сообщество + память + кино.

**Два регистра тона** (переключаются по разделам, палитра покрывает оба):
- **Тёплый-живой** — Главная, Новости, Проекты, Свитино ТВ, Фильм. Зелёный + терракота, светлые секции, живые фото.
- **Торжественный-строгий** — Герои (1941–1945), История/Архив. Графит + камень + ч/б, сдержанность, достоинство.

## 2. Фирменные мотивы (визуальная подпись)

- **Габион** (камень в стальной сетке) — главный материал Свитино: въездной знак «СВИТИНО», стены, основания памятников (в 8 из 13 фото). Используется как: фон-текстура секций, рамки/бордеры карточек с намёком на сетку, разделители.
- **Вордмарк «СВИТИНО»** из габионных букв — фактический логотип посёлка. Логотип сайта: `SVITINO.TV`.
- **Два дуба + полоска киноплёнки** — эмблема кино-истории (с мемориальной таблички). Воссоздать как SVG-иконку для кино-раздела и сквозного мотива.
- **Киноплёнка** (перфорация, кадры) — мотив для лент новостей, галерей, разделителей.
- **Ч/б архивная фотография** — для исторических материалов («тогда»), цветные живые фото — для «сегодня».

## 4. Типографика

- **Заголовки:** гуманистический гротеск с характером — `Manrope` (700/800). Укоренённый, современный, тёплый. Альт: `Onest`.
- **Текст:** чистый гуманистический сан — `Inter` / системный. 16px база, line-height 1.6, длина строки 65–75ch.
- **Акцент / вордмарк:** `SVITINO.TV` — 800, letter-spacing `-0.02em`. `.TV` цветом `--sky` или `--terracotta`.
- **Раздел памяти:** заголовки `1941–1945` — крупно, разрядка `+0.05em`, сдержанно (как гравировка на плитах).
- `text-wrap: balance` на h1–h3; потолок hero ≤ 6rem; tracking заголовков ≥ -0.04em.

## 5. Фотостиль

- **Документальные, живые, непостановочные** кадры (люди в среде, события, природа). Не сток.
- Кинематографичный естественный свет, мягкий, чуть прохладный.
- Габион как фон/связующая фактура.
- Архив — ч/б или лёгкая сепия; «сегодня» — цвет.
- Кадрировать крупно, давать «дышать».

## 6. UI-приёмы

- Шапка/футер — графит (техно-«рамка»), контент — на бумаге (тёплое наполнение).
- Карточки: на бумаге/песке, тонкая `--line`, hover — подъём + тень + бордер `--green`/`--stone`.
- Кнопка-primary: `--green` (или `--terracotta` для «действия»); outline для вторичных.
- Метка «● Прямой эфир» — `--sky`/`--crimson`, пульсация.
- Скругления 8–12px; анимации 180–260ms; видимый `:focus`.
- Габион-текстура — через CSS-паттерн или фото `sign-*` / `archive-gallery` как фон с тёмным оверлеем для читабельности.
