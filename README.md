# SVITINO.TV — Новая версия

## Что изменилось

- **Стиль**: тёмный документальный (#1c1c19), серифная типографика (PT Serif), паттерн габиона как текстура
- **Палитра**: уголь · бумага · янтарь · камень · ель
- **Главная**: героический экран с архивной сеткой фона, габионный mesh-оверлей, strip-метаданные (Сезон / Формат / Статус) из v2
- **История места**: цитата с мемориальной таблички деревни на главной
- **Шрифты**: PT Serif (заголовки), Inter (интерфейс) — оба Google Fonts, кириллица

## Структура

```
index.html         — главная
tv.html            — Свитино ТВ
heroes.html        — герои
film.html          — фильм
news.html          — новости
projects.html      — проекты
contacts.html      — контакты
css/style.css      — все стили
js/nav.js          — бургер + активная ссылка
```

## Деплой

Загрузить содержимое папки в корень хостинга svitino.tv.
